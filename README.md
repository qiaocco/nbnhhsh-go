# 「能不能好好说话？」 的 Golang 版本

## 安装

#### 使用go get安装

```bash
go get https://github.com/qiaocco/nbnhhsh-go
```

#### 手动安装

从 [releases](https://github.com/qiaocco/nbnhhsh-go/releases) 下载最新发布版本, 解压可执行文件到 /usr/bin


## 例子

```shell

hhsh "s姐yyds, alal"

```
